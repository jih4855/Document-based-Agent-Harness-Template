# 런타임 에이전트 템플릿

이 폴더는 서브에이전트 handoff 템플릿을 저장합니다.

이 파일들은 상주 에이전트가 아닙니다. 위임을 지원하는 도구에서 메인이
런타임 서브에이전트를 만들 때 사용하는 프롬프트와 운영 계약입니다.

## 원칙

- 하나의 서브에이전트는 하나의 업무만 처리합니다.
- 필요한 파일과 결정만 전달합니다.
- 절차는 skill에, handoff 규칙은 runtime template에 둡니다.
- 서브에이전트는 문서화된 규칙을 따르고 새 취향을 만들지 않습니다.
- 막히면 blocker를 보고하고 메인이 멘토링하거나 직접 인수합니다.

## 권장 템플릿 구조

```markdown
# example-agent

## 목적

이 에이전트가 처리할 반복 업무.

## 읽을 파일

- `context/tacit.md`
- 관련 스킬 문서
- 작업별 원본 파일

## 허용 작업

- scoped action 1
- scoped action 2

## 금지 작업

- 무관 workflow
- 광범위한 정리
- 명시 승인 없는 되돌리기 어려운 작업

## 출력 계약

모든 서브에이전트가 확장해야 할 표준 계약은 `context/agent_index.md`를
참조합니다.
```

# 메모리

이 파일을 `context/memory.md`로 복사해 실제 프로젝트 메모리로 사용합니다.

저장소가 공개되어 있거나 공유 범위가 넓다면 사적인 live memory를 커밋하지 않습니다.

## 최근 맥락

- YYYY-MM-DD: 템플릿으로 초기 하네스를 생성함.

## 결정 사항

- YYYY-MM-DD: portable root instruction 파일로 `agent.md`를 사용하기로 함.
- YYYY-MM-DD: live state는 `context/memory.md`와 `context/task.md`에 두기로 함.

## 다음 세션 참고

- 먼저 `agent.md`를 읽습니다.
- 현재 작업은 `context/task.md`에서 확인합니다.
- 프로젝트별 선호와 규칙은 `context/tacit.md`에 둡니다.
